# USA and Mexico Hours Worked EDA

## Overview

First Exploratory Data Analysis (EDA) project using USA and Mexico working hours datasets.

The goal of this project was to explore the datasets, identify data quality issues, handle missing values, and prepare the data for further analysis.

## Dataset

- USA Hours Worked Dataset
- Mexico Hours Worked Dataset

## Approach

The analysis included:

- Loading and exploring the datasets
- Checking dataset dimensions and structure
- Identifying missing values
- Cleaning and handling missing data
- Combining datasets for comparison
- Preparing the data for analysis

## Data Cleaning Steps

The project focused on:

- Detecting missing values
- Replacing missing values where appropriate
- Removing unnecessary missing entries
- Combining USA and Mexico datasets into one dataset
- Adding country information to distinguish between datasets

## Goal

The main goal was to understand the dataset structure and perform basic EDA operations while improving data cleaning skills.

## Libraries Used

- Pandas
- NumPy
