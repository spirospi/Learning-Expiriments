# California Housing Price Prediction

## Approach

I experimented with multiple regression models to compare their performance:

- Linear Regression (baseline model)
- Decision Tree Regressor
- Support Vector Regression (Linear and RBF kernels)
- Random Forest Regressor

The initial Linear Regression model showed limited predictive performance.
The Decision Tree model suffered from overfitting, while the SVR models provided better results.

Overall, Random Forest achieved the best performance among the tested models.

I experimented with all these models before completing the final exercises from the end of the chapter.

Additionally, I performed error analysis as suggested in the Hands-On Machine Learning book to better understand model behavior and prediction errors.
