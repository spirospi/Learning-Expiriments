# Machine Learning For Everybody

Projects completed while following Kylie Ying's Machine Learning For Everybody tutorial.

The goal of these projects was to review and reinforce core machine learning concepts, including classification, regression, and unsupervised learning.

## Projects

### 1. MAGIC Gamma Classification

- Task: Binary Classification
- Dataset: MAGIC Gamma Telescope Dataset
- Goal: Classify gamma-ray events from hadron events

- Models:
  - K-Nearest Neighbors (KNN)
  - Gaussian Naive Bayes
  - Logistic Regression
  - Support Vector Classifier (SVC)

- Techniques:
  - Data preprocessing
  - Feature scaling with StandardScaler
  - Handling class imbalance with RandomOverSampler


### 2. FCC Bikes Regression

- Task: Regression
- Dataset: Bike Sharing Dataset
- Goal: Predict bike-sharing demand

- Models:
  - Linear Regression
  - Neural Network Regression (TensorFlow)

- Techniques:
  - Data preprocessing
  - Feature scaling with StandardScaler
  - Data visualization


### 3. FCC Seeds Unsupervised Learning

- Task: Unsupervised Learning
- Dataset: Seeds Dataset

- Methods:
  - K-Means Clustering
  - Principal Component Analysis (PCA)

- Libraries:
  - Pandas
  - NumPy
  - Matplotlib
  - Seaborn

- Goal:
  - Identify patterns and group similar data points without using labeled data
