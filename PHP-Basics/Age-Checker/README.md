# Age Checker

## Description

A simple PHP web application that checks whether a person has the right to vote based on their age.

The user enters their age through an HTML form, and the PHP script processes the input and displays whether the user is eligible to vote.

## Features

* User input through an HTML form
* Age validation
* Checks if the age is a valid positive number
* Displays voting eligibility based on age

## Concepts Practiced

* PHP variables
* HTML forms
* POST method
* Conditional statements (`if/else`)
* Input validation
* Basic server-side processing

## Technologies

* PHP
* HTML
