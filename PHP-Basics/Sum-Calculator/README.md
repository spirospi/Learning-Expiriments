
# Sum Calculator

## Description

A simple PHP program that calculates the sum of numbers from 1 to 10.

The program uses a loop to add each number in the range and displays the final result on a web page.

## Features

* Calculates the sum of numbers from 1 to 10
* Displays the result dynamically using PHP

## Concepts Practiced

* PHP variables
* For loops
* Arithmetic operations
* Basic PHP output
* Embedding PHP inside HTML

## Technologies

* PHP
* HTML
