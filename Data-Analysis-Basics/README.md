# Reddit API Analysis

## Overview

Small data analysis project using Reddit API data.

The goal of this project was to collect Reddit comments related to programming and analyze subreddit frequency using Python.

## Approach

The project includes:

- Creating a function to request data from the Reddit API
- Collecting programming-related comment data
- Converting API responses into a Pandas DataFrame
- Exploring subreddit frequency
- Visualizing the results with a bar chart

## Data Processing

The workflow included:

- Sending API requests with specific parameters
- Processing JSON responses
- Creating a structured dataset using Pandas

## Libraries Used

- Requests
- Pandas
- Matplotlib

## Goal

Practice working with APIs and applying basic data analysis techniques using Python.
