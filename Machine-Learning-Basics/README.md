# Machine Learning Basics

## Overview

Collection of small machine learning experiments created while learning and revisiting fundamental machine learning concepts.

The goal of these experiments was to understand basic machine learning algorithms, preprocessing techniques, and model evaluation.

## Projects

## 1. Linear Regression Basics

**Task:** Regression

**Description:**
A simple experiment using generated data to understand the fundamentals of linear regression.

**Model:**
- Linear Regression

**Concepts Covered:**
- Creating synthetic data
- Data visualization
- Model fitting
- Understanding coefficients and intercept


---

## 2. Iris KNN Classification

**Task:** Multi-class Classification

**Dataset:**
- Iris Dataset

**Model:**
- K-Nearest Neighbors (KNN)

**Concepts Covered:**
- Train/Test split
- Feature normalization
- Data visualization
- Understanding KNN algorithm

**Additional Implementation:**
Implemented KNN from scratch:
- Euclidean distance calculation
- Nearest neighbor selection
- Majority voting


---

## 3. Iowa House Price Regression

**Task:** Regression

**Dataset:**
- Iowa House Prices Dataset

**Model:**
- Random Forest Regressor

**Concepts Covered:**
- Feature selection
- Train/validation split
- Model evaluation using MAE
